1: Vercel AI SDK 3.0

通过React服务器组件实现了将文本和图像提示转换为React UI，并简化了设计工程流程

Vercel AI SDK 3.0是一个开源的生成式UI技术，旨在将文本和图片提示转换为React UI，并简化设计工程过程。这一技术可以让开发者利用React服务器组件(RSC)将LLMs的响应与流式React服务器组件关联起来。通过这一更新，开发者可以实现更丰富、基于组件的用户界面，让用户在搜索体验中直接渲染组件，更轻松地解释代理执行，让用户了解幕后的魔法。AI SDK 3.0还支持在LLMs的响应中使用实时数据，以及调用OpenAI兼容的功能或工具，从而实现更丰富和交互性更强的应用程序。这一更新还使得将人工智能集成到应用程序中变得更加简单，让应用程序更具交互性和响应性，而无需牺牲性能。

地址：https://github.com/vercel/ai

2: ChatGPT-PLUS

AI 助手全套开源解决方案，自带运营管理后台，开箱即用。

ChatGPT-PLUS 基于 AI 大语言模型 API 实现的 AI 助手全套开源解决方案，自带运营管理后台，开箱即用。集成了 OpenAI, Azure, ChatGLM,讯飞星火，文心一言等多个平台的大语言模型。集成了 MidJourney 和 Stable Diffusion AI绘画功能。

地址：https://github.com/yangjian102621/chatgpt-plus

3: D-ID Agents

高效、灵活、可定制化的虚拟数字人制作平台

D-ID是一家数字人制作平台，主要提供数字人生成视频和可交互数字人的实时交流功能。D-ID支持上传自己的知识库，利用检索增强生成（RAG）技术，使数字人能够提供更准确、更精细的信息。此外，D-ID提供了10+种中文方言和多种语言的口音，为视频制作提供了多样性，同时也可以实现可交互数字人的实时交流。D-ID在企业营销推广、数字教师、数字客服等领域具有广泛的应用前景和市场需求。

地址：https://studio.d-id.com/agents

4: Jan

开源的ChatGPT替代品，可以100%离线运行在您的电脑上。

Jan是一个开源的类ChatGPT程序，可以完全离线在你的电脑上运行。无论是个人电脑还是多GPU集群，Jan都能支持运行，这意味着它支持包括Nvidia GPUs、Apple M系列芯片、Apple Intel处理器、Linux Debian系统和Windows x64系统等各种不同的硬件架构。这使得Jan成为了一个非常灵活和通用的工具。

地址：https://github.com/janhq/jan

5: Microsoft Copilot for Finance

微软为财务专业人士提供的帮助工具

Microsoft Copilot for Finance是微软为财务专业人士推出的一项工具，旨在加速业务影响的实现。它通过提供洞察力，帮助减少手动、重复工作所花费的时间。Copilot能够加速对财务洞察力的影响，通过行动导向的建议、主动异常检测以及专业提示和指导来促进增长。此外，它还可以优化财务流程，减少成本，提高效率，将数据转化为洞见，加速决策时间。通过Copilot，可以在Excel、Outlook和Teams等微软生产力工具中无缝工作，而无需切换应用程序，同时也能够连接到现有的财务数据源，包括ERP等。Copilot还具备安全、合规和隐私政策，因为它会自动继承在Microsoft 365中设置的政策。

地址：https://www.microsoft.com/en-us/microsoft-copilot/microsoft-copilot-for-finance
