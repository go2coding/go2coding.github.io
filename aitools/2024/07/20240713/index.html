✨ 1: AiEditor

AiEditor是一款基于AI的多功能富文本编辑器，支持多种前端框架及灵活配置。

AiEditor 是一个由 AI 驱动的新一代富文本编辑器，基于 Web Component 技术，支持 Vue、React、Angular 等几乎所有前端框架，同时兼容 PC 和移动端设备，并提供明暗两种主题。其主要特点包括灵活的配置和易于使用，允许开发者轻松构建各种文字编辑应用。

AI驱动：完全由 AI 驱动，支持对接任何大型模型，包括私有模型，用户可以使用自己的私有大型模型 API 密钥。 友好易用：基于 Web Component 开发，易于集成到任何主流前端框架中，采用 LGPL 开源协议。 多个版本：除了开源版本，还提供功能更强大的商业版本，不限制用户数量和应用数量。

功能特性包括基础文字编辑、增强功能（如撤销、重做、格式刷）、附件支持、代码编辑、表格操作、Markdown 支持和多种 AI 功能（如 AI 续写、AI 优化、AI 校对、AI 翻译）等。此外还具有国际化、多主题和移动端适配等特性。

官方主页和更多信息请访问：https://aieditor.dev

地址：https://github.com/aieditor-team/AiEditor/blob/main/readme.md

✨ 2: Mira

Mira 是腾讯推出的用于长时视频生成的初步框架。

Mira是腾讯PCG ARC实验室的一个新项目，旨在探索高质量、长时长的视频生成，类似Sora。与现有的文本到视频（T2V）生成框架相比，Mira具有几项显著的优势：

扩展的序列长度：大多数框架只能生成2秒（16帧）的视频，而Mira设计生成的序列更长，可以达到10秒、20秒甚至更多。 增强的动态效果：Mira能够生成具有丰富动态和复杂动作的视频，超越当前视频生成技术中较为静态的输出。 强大的3D一致性：即使在动态的场景和物体交互过程中，Mira也能保持物体的3D完整性，避免明显的失真。

MiraData目前包括两个场景：游戏和城市/风景探索，未来将添加更多场景并提高数据集的质量。

github 地址：https://github.com/mira-space/MiraData

地址：https://mira-space.github.io/

✨ 3: Knowledge Graph Builder App

Knowledge Graph Builder App 是一个将非结构化数据转换为结构化知识图谱的应用。

Knowledge Graph Builder App 是一款应用程序，旨在将非结构化数据（如PDF文件、文档、文本、YouTube视频、网页等）转换为存储在Neo4j中的知识图谱。该应用利用大语言模型（如OpenAI、Gemini等）的强大功能，从文本中提取节点、关系及其属性，并使用Langchain框架创建结构化的知识图谱。

知识图谱创建：使用大语言模型将非结构化数据转换为结构化的知识图谱。 提供架构：可以提供自定义架构或使用设置中的现有架构来生成图谱。 查看图谱：可在Bloom中查看特定来源或同时查看多个来源的图谱。 数据交互：通过对话查询的方式与Neo4j数据库中的数据交互，并检索查询响应的元数据。

地址：https://github.com/neo4j-labs/llm-graph-builder

✨ 4: Ollama App

Ollama App 是一款现代易用的客户端，为您提供本地网络隐私保护的极佳体验。

Ollama App 是一个现代且易用的 Ollama 客户端，提供良好的用户体验，同时确保所有数据都在本地网络中私密处理。该应用程序并不在设备上托管 Ollama 服务器，而是连接到一个预设的 Ollama 服务器并使用其 API 端点。

聊天互动：Ollama App 提供一个简洁的聊天界面，用户可以选择不同的模型来进行聊天。该应用程序支持多模态输入，可以通过图片和文本进行互动。 自定义配置：用户可以自定义应用的许多设置，如主机地址、行为模式、界面外观等。 数据备份与导出：应用支持将聊天内容导出保存，用户可以备份数据以防丢失或者在多个设备间同步。 多语言界面：应用内置多语言支持，如英文和德文，用户可以根据需要选择合适的语言界面。 语音交互：Ollama App 还提供实验性的语音模式，支持语音输入进行文本转换。

地址：https://github.com/JHubi1/ollama-app

✨ 5: AI Artifacts

AI Artifacts是一个开源应用，使用Anthropic的AI模型和E2B的SDK，来安全执行AI生成的代码。

AI Artifacts 是一个开源应用，基于 Anthropic 的 Artifacts UI 和 Claude 聊天应用。该应用使用了 E2B 的 Code Interpreter SDK 来执行 AI 生成的代码，支持安装库、运行 shell 命令、执行 Python、JavaScript、R 和 Next.js 应用等。

地址：https://github.com/e2b-dev/ai-artifacts
