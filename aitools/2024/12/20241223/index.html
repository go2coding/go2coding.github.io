✨ 1: Genesis

Genesis是一个多功能物理平台，适用于机器人、具身智能和物理AI应用，提供高效的仿真和数据生成。

⭐️ 极高人气，开源仅两天便吸引了 15.8K 用户 ⭐️

Genesis 是一款专为机器人和人工智能领域打造的物理模拟平台，具有卓越的性能表现。

卓越性能 惊人速度：在 RTX 4090 显卡上，模拟 Franka 机械臂的速度可达每秒 4300 万帧，远超实时需求，快了 43 万倍。 跨平台支持：兼容 Linux、MacOS 和 Windows，能够在 CPU、NVIDIA GPU、AMD GPU 以及 Apple Metal 上运行。

强大的物理引擎 多种物理求解器整合：支持刚体、流体、可变形体等多种物理模拟。 丰富的材料模型：可模拟刚体、液体、气体及可变形物体等多种材料。 多样化的机器人类型：支持机械臂、足式机器人、无人机及软体机器人等多种形式。

用户友好性 简单安装：只需通过 pip install genesis-world 即可轻松安装。 直观的 API 设计：关注用户体验，便于开发者使用。 多种文件格式支持：兼容 MJCF、URDF、obj、glb 等多种格式。

地址：https://github.com/Genesis-Embodied-AI/Genesis

✨ 2: clickclickclick

ClickClickClick是一个框架，利用任何大型语言模型实现安卓手机和计算机的自主操作。

ClickClickClick 是一个框架，旨在通过任何大型语言模型（LLM，既可以是本地的也可以是远程的）来实现自动化的安卓手机和计算机操作。该框架的目标是提供一种便捷的方式，让用户通过自然语言指令实现复杂的任务。

ClickClickClick 提供了一个强大的自动化工具，可以在多种场景中提高工作效率，减少操作步骤，使用户能够以更自然的方式与设备交互。

地址：https://github.com/BandarLabs/clickclickclick

✨ 3: AniDoc

AniDoc是一个简化动画创作过程的工具，可以精准为手稿上色，并保持高保真度。

AniDoc 是一个旨在简化动画创作过程的工具，能够基于角色设计参考高保真地为一系列草图上色，即使这些草图在姿势和比例上有显著差异。它利用先进的图像处理技术，使用户能够快速生成动画效果，甚至可以适应不同的草图和角色设计。

地址：https://github.com/yihao-meng/AniDoc

✨ 4: picotron

Picotron是一个简洁易用的Llama-like模型预训练框架，支持多维并行，适合学习和实验。

Picotron是一个简约且具高度可扩展性的预训练Llama类模型的代码库，旨在通过4D并行性（数据、张量、管道和上下文并行）实现高效的模型训练。它的设计专注于简单易懂，适用于学习和实验，特别适合想要深入理解模型训练过程的用户。

地址：https://github.com/huggingface/picotron

✨ 5: openai-realtime-twilio-demo

OpenAI Realtime API与Twilio结合，构建电话AI助手，实现实时通话和消息转发功能。

OpenAI Realtime API与Twilio的快速入门演示项目（openai-realtime-twilio-demo）结合了OpenAI的实时API与Twilio的电话呼叫功能，旨在构建一个AI电话助手。

地址：https://github.com/openai/openai-realtime-twilio-demo
