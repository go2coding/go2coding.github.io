✨ 1: gemma-3

Gemma 3是Google基于Gemini 2.0研发的轻量级开源模型系列，可在单GPU或TPU上运行，性能强大。

Gemma 3 是 Google 基于 Gemini 2.0 技术构建的最新一代开源模型系列，旨在提供高性能和易用性，特别适合在单个 GPU 或 TPU 上运行。它的主要特点包括：

卓越性能: 在同等规模的模型中表现出色，超越了 Llama-405B 和 DeepSeek-V3 等模型。 多语言支持: 支持超过 140 种语言，为全球用户提供服务。 高级推理能力: 具备强大的文本和视觉推理能力，能够处理图像、文本和短视频。 超大上下文窗口: 拥有 128k token 的上下文窗口，可以处理大量信息。 功能调用支持: 支持功能调用和结构化输出，方便自动化任务和构建智能代理。 量化模型: 提供量化版本，减少模型大小和计算需求，提高性能。 安全特性: 经过严格的安全测试，并推出 ShieldGemma 2 作为图像安全检测器，用于识别危险内容。 易于集成: 可以与 Hugging Face Transformers、Ollama、JAX、Keras、PyTorch 等常用工具无缝集成。 多种部署方式: 支持 Vertex AI、Cloud Run、Google GenAI API、本地环境等多种部署平台。

地址：https://blog.google/technology/developers/gemma-3/

✨ 2: AI-Researcher

AI-Researcher利用LLM自动执行科研流程，从文献回顾到论文撰写，旨在加速科学发现并革新研究范式。

AI-Researcher 是一个革命性的全自动科学发现系统，旨在重塑传统的研究范式。它通过集成先进的AI代理，实现了从概念到出版的端到端研究自动化，显著加速了科学创新。

核心能力：

全自动研究流程： 从文献综述、研究思路生成、算法设计与实现、算法验证与优化、结果分析到论文撰写，全流程自动化。 智能研究助手： 接受用户输入的研究思路或相关论文，自动进行研究设计、实验、分析和撰写报告。 多LLM支持： 可以灵活选择和集成多种主流大型语言模型（LLM）供应商的服务，如Claude、OpenAI、Deepseek等。 全面的评估基准： 提供一套用于评估AI研究人员学术能力的标准化评估框架，确保评估的彻底性和可靠性。 即开即用： 经过最小化设置即可立即使用，用户可以快速接入并加速研究过程。

地址：https://github.com/HKUDS/AI-Researcher

✨ 3: Playwright MCP Server

Playwright MCP Server是一个使用Playwright的MCP服务器，让LLM能自动化浏览器操作，截图和执行JavaScript。

Playwright MCP Server是一个基于Playwright的Model Context Protocol (MCP)服务器。它的主要功能是提供浏览器自动化能力，让大型语言模型 (LLM) 能够与网页进行交互，例如：

访问和操作网页: LLM 可以通过这个服务器来访问特定的网页，并在网页上执行操作，例如点击链接、填写表单等。 截图: LLM 可以指示服务器截取网页的屏幕截图。 执行JavaScript代码: LLM 可以在网页上执行 JavaScript 代码，实现更复杂的操作。

地址：https://github.com/executeautomation/mcp-playwright

✨ 4: BlenderMCP

BlenderMCP通过MCP协议连接Blender和Claude AI，实现AI辅助3D建模、场景创建和操控等功能。

BlenderMCP 是一个连接 Blender 和 Claude AI 的工具，它通过模型上下文协议 (MCP) 让 Claude AI 能够直接与 Blender 交互并控制 Blender。简单来说，你可以用 Claude AI 的自然语言提示来辅助你在 Blender 中进行 3D 建模、场景创建和修改。

地址：https://github.com/ahujasid/blender-mcp

✨ 5: Pocket Flow Framework

Pocket Flow是一个快速构建企业级AI系统的框架，基于嵌套有向图，模块化且不依赖特定厂商。

Pocket Flow Framework 是一个用于快速构建企业级 AI 系统的框架，它的核心理念是基于嵌套有向图 (Nested Directed Graph) 的抽象，将复杂的 AI 任务分解为多个可重用的 LLM 步骤，并通过分支和递归实现类似 Agent 的决策能力。

地址：https://github.com/The-Pocket-World/Pocket-Flow-Framework
