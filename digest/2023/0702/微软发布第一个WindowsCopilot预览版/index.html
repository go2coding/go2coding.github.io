6月30日，微软发布了Windows 11 Insider Preview Build 23493。该版本最大亮点是提供了第一个Windows Copilot预览。

微软表示，第一个Windows Copilot预览版重点关注集成UI体验 ，更多功能将在未来的预览版中推出。要想使用该版本，用户必须在开发频道中拥有Windows Build 23493 或更高版本，以及 Microsoft Edge 115.0.1901.150 或更高版本。

Windows Copilot预览功能演示

如何使用Windows Copilot

使用方法很简单，只需单击任务栏上的【PRE】图标或 WIN + C，即可启动Windows Copilot。

Windows Copilot 将停靠在右面的侧栏，不会与桌面内容重叠，并且将在打开的应用程序窗口旁边运行，使得用户可以随时与 Windows Copilot 进行智能交互。

例如，帮我看看最近一周天气如何？可以帮我总结一下这份PDF报告的重点摘要吗？有哪些最新的NBA消息？Copilot都能快速提供实时信息。

在第一个Windows Copilot预览版中，用户可以执行以下操作：

  • 切换深色模式；

  • 开启请勿打扰；

  • 截图；

  • 总结网站内容（Edge浏览器中的“活动”选项卡）；

  • 支持创建文本内容，例如，帮我写一个关于住在月球上的狗的故事；

  • 帮我生成一张宁静的锦鲤鱼塘和盛开荷花的图片。

用户在使用的过程中遇见任何问题，可以点击Windows Copilot右上角进行提交反馈，帮助微软进一步优化功能。

此外，Windows Copilot 目前提供一组有限的 Windows 设置插件；但目前不支持第三方插件。随着时间的推移，微软将继续完善Windows Copilot添加更多的功能。

什么是Windows Copilot

早在今年5月，微软在举办的全球“Build大会”上宣布，将生成式AI和Bing Chat集成在Windows中推出——Windows Copilot。

Copilot是一种类ChatGPT产品，用户通过自然语言就能生成文本、代码、查询信息等。微软将其集成在Windows 11中并且可以与Bing Chat、插件等一起使用，成为你的专属“智能秘书”。

用户通过Copilot可以做很多事情，例如，将PDF直接发送给Copilot，帮我总结一下PDF的核心内容；帮我生成一份关于自然科学的演讲稿，要求轻松、活泼一些；我想去伦敦旅行，能提供一个详细的计划吗？

回顾历史，1985年11月，微软发布了Windows 1.0版本终结了IBM的MS-DOS操作系统，开启了PC图形交互时代。现在，微软通过在Windows 11中集成Copilot，开启全新智能交互系统时代。
