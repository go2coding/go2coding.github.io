# 结语

本文提供的这些提示词并非在每种情况下都有效。DALL-E 3 会根据图片的内容决定如何添加文本。如果要添加的效果跟图片的内容或风格搭配不起来的话，文本会显示不出来。所以在添加文字效果前，不妨想象一下，这个效果合理和美观吗？

还有就是 DALL-E 3 在生成比较长的文字时经常会犯错，所以尽量还是选择比较简短的文本。

DALL-E 3 的可能性还有很多，所以请不要局限于文中总结的效果，尽情地发挥你的想象力吧！
