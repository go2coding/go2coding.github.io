ChatGPT代码解释器：解锁十一种令人惊叹的玩法


  • By AiBard123
  • July 21, 2023 - 2 min read



作者： 叶赛文 来源： 叶赛文

openai在上周发布了一个新功能：代码解释器（Code Interpreter），它是ChatGPT 有史以来最强大、最有用的更新之一。

这个新功能的名称里有“代码”二字，其实不要被它吓唬到了，不是程序员，也能用它。

简单地理解，它就是你雇佣的一个初级程序员 。你让它干嘛，它就干嘛。

如何打开代码解释器？

必须要有ChatGPT plus，才能使用代码解释器。

在ChatGPT的聊天界面，点击左下角的setting（设置），然后点击beta里的代码解释器开关。

玩法1：为Midjourney描述图像

第一步：上传图片，

并添加提示词（仅供参考）“Describe this image to me ”

ChatGPT会生成这张图片的描述。

第二步：让ChatGPT生成Midjourney或者dall-e提示词

输入提示词：Write an image generation prompt for midjourney or Dall-E that recreates this image from Al（为 midjourney 或 Dall-E 编写图像生成提示，从 Al 重新创建此图像）

第三步：复制、粘贴提示词

复制、粘贴提示词，把它放到Midjourney里。

玩法2：生成二维码

参考提示词：Create a working QR code that opens this website when scanned: http://yesaiwen.com (创建一个有效的二维码，扫描后可打开此网站：http://yesaiwen.com)

再也不用为创建二维码而发愁了。

玩法3：项目管理

项目管理很麻烦。

如果有人做过甘特图，就知道它有多么耗时。

而现在有了ChatGPT的代码解释器，输入文本，就可以让其生成甘特图。

参考提示词：

Create a gantt chart or project roadmap for the following（为以下内容创建甘特图或项目路线图）

Create a gantt chart or project roadmap for a project called: school of surgery（项目名称：外科学校）

The timeline should run from now until the end of 2023.（时间表应从现在开始一直持续到 2023 年底）

Key deadlines are:（主要截止日期是：） 1.Collect robotic and laparoscopic video footage 5/8/23（收集机器人和腹腔镜视频片段5/8/23） 2.Send email to contributors identifying Spr and SHO to co-ordinate on project from August21/7/23（向贡献者发送电子邮件，确定 Spr 和 SHO，以便从 2023 年 8 月 21 日起协调项目）

可以让代码解释器生成csv文件，然后导入到类似Asana的工具里，就会生成实用的甘特图。

玩法4：复习间隔表

代码解释器适用于目前正在学习的任何人（无论是在职业生涯还是在学校）。

隔一段时间，主动复习相关的内容，可以消除遗忘曲线的影响。

但是，对于新手来说， 创建间隔时间表和学习时间表可能非常耗时，而且有点棘手。

使用ChatGPT的代码解释器，可以帮我们快速指定间隔表。

首先需要下载复习间隔表，通过代码解释器，把它上传到ChatGPT里。

参考提示词：

使用复习时间表的大纲来制定我的医学期末考试的间隔时间表。

我的考试日期是 2024 年 7 月 12 日。我需要涵盖完整的 USMLE/医学期末课程。填充我需要涵盖的所有主题，包括心脏病学、呼吸学、内分泌学等。使用与 supermemo 使用的标准间隔时间表类似的标准间隔时间表。请遵循所提供文档的格式。

最后，ChatGPT会生成一个间隔复习时间表。

玩法5：文件转换

这是一个非常实用的玩法。

日常生活中，我们经常需要转换文件。

ChatGPT的代码解释器，支持各类文件的转换，比如：视频转音频，pdf转Word，不同视频格式间的转换，等等。

玩法6：复杂的数学方程

如果您正在学习数学或物理等学科，或者您在工程等行业工作，需要进行大量复杂的数学计算和方程，那么代码解释器将非常有用。

代码解释器可以运行一些相当复杂的数学方程并生成图表和其他东西。

如果您有大量的数字数据集或来自研究的数据，您需要运行深入的数学方程来得出结果，例如计算 P 值或特定干预措施（如医学）的影响，那么您可以通过 ChatGPT 来完成此操作，而不必使用付费公式方程软件。

玩法7：文件问答

这是一个非常实用的功能。

当我们遇到一个大的文档（例如pdf版的公司制度，openai的官方文档等），或者是大的数据集时，要在这里面找一个问题的答案。

一页一页地阅读，太浪费时间。

以前需要用chatpdf等网站，或者是第三方插件来操作。

现在可以直接使用代码解释器来节省大量时间。

直接把文件上传给代码解释器，然后就可以要求ChatGPT，根据上传的文档回答问题了。

玩法8：数据可视化

当我们需要对数据做分析的时候，比如说销售数据、健康数据等等，可以把它们直接上传到ChatGPT代码解释器里。

然后要求它分析数据，创建各种图表。

还能创建让人一目了然的词云。

玩法10：分析代码及从头开始从头开始写代码

可以将代码打包成zip，上传到代码解析器里。然后由 ChatGPT 分析和评论代码，并给出优化建议。

ChatGPT的代码解释器（Code Interpreter）， 不仅仅可以处理代码分析。实际上，您可以要求它从头开始创建代码，就像使用普通 ChatGPT 一样。

玩法11：视频处理

代码解释器可以对视频进行一些基础的编辑，比如翻转视频，甚至更改视频文件格式，或者如果我们有带音频的视频，则直接提取音频。

代码解释器的不足

虽然代码解释器确实很棒，但它仍处于测试阶段，并且并不总是完全符合我们的要求。

比如上传文件时，文件保存在临时存储区，随时会丢，需要重新上传。

或者忘记之前写的代码。

但不管怎么说， Code Interpreter是一个非常出色的工具，特别适用于处理大型数据集，将在未来节省我们大量时间。
