又到了一年一度的毕业季，许多同学开始忙着写简历、找工作、到各个公司参加面试。

如何写出一份排版整洁、结构清晰的个人简历，可谓是大多数人在面试求职过程中必须思考的诸多问题之一。

最近，我在 GitHub 上看到不少实用且功能强大的开源简历项目，今天在这里做下分享。

希望通过本期推荐的几款工具，帮助大家更好的制作一份出色、独特的个人简历，在面试过程中让面试官眼前一亮，增加好感。

简历自动生成 & 解析器

这两天，一款名为 OpenResume 的开源项目登上了 GitHub 热榜。

该项目可支持根据用户提供的简历文本，自动生成排版精美的 PDF 个人简历，或解析已有的 PDF 简历文件。

在线试玩：https://open-resume.com

GitHub：https://github.com/xitanggg/open-resume‍

在信息输入时，预览页面会实时刷新，让你立马可以看到更改后的效果展示。

设计风格偏现代化，可自动设置简历字体、大小、边距、项目符号格式，确保所有风格统一，让内容看起来更为清晰。

项目可完全托管在本地离线运行，无需注册，开箱即用。在使用过程中，你也不用担心个人隐私泄露。

如果你之前已经写过简历，还可以用这款工具直接导入你原有的简历 PDF 文件，重新修改之后，再用它生成新的简历。

经该工具设计后的个人简历，已有不少成功拿到了知名互联网公司，如何 Dropbox、Google、MetaAI 等企业的 Offer，因此在简历设计与排版上，你可以相信它的专业程度。

功能强大的简历生成器

目前互联网上有不少简历排版工具，甚至是一些 Markdown 编辑器，也支持直接导出成 PDF，快速生成简历。

不过，却几乎没有一款工具，可以针对简历生成这项硬性需求，把用户体验与使用细节做得尽善尽美的。

除了这一款，在 GitHub 上开源的简历生成器：Reactive Resume 。

GitHub：https://github.com/AmruthPillai/Reactive-Resume

该工具可通过各种配置，让创建、更新、共享简历的过程变得更加简单便捷，并且免费开源，支持中文！

你可以一次性创作多份简历，并为每个简历生成特定链接，导出生成 PDF 文件，随时随地进行分享。

另外，它还支持对简历外观进行灵活定制，颜色、主题、布局任意变动。暗黑模式与明亮模式，快速切换。

所有的内容与外观变动，均所见即所得，无需等待与加载更新，整个过程极为流畅。

当然，如果你觉得简历内容更新维护比较麻烦，也可以通过下面这个网站，直接使用 JSON 来撰写简历。

地址：https://jsonresume.org/

习惯使用 LinkedIn 来保存与更新自己个人简历的同学，也可以将平台上的数据，一键导入到 Reactive Resume 工具。

个人简历博客

现在有不少同学，习惯用个人博客来托管个人简历，以便在学习与工作的过程中，及时更新自己的产出，以便获得更多工作机会。

如果你也属于此列，那我在这里给你推荐一个 GitHub 开源项目：Online CV 。

GitHub：https://github.com/sharu725/online-cv

这是一个支持响应式设计的 Jekyll 个人博客主题，可以将你的简历信息上传到 GitHub。

利用 GitHub Pages + Jekyll，即可快速部署一个简洁美观的个人简历博客，用于对外展示你的简历信息。

该主题支持 6 种不同风格的主题配色，用户可根据个人喜好随时调整。

项目本地部署也比较简单，在确保已安装 Git 与 Ruby 环境的情况下，使用下面 3 条命令，即可快速完成本地项目部署。

# 克隆 GitHub 项目  
git clone https://github.com/sharu725/online-cv.git  
  
# 安装项目依赖（Ruby Gems）  
bundle install  
  
# 在本地运行 Jekyll 服务器  
bundle exec jekyll serve

部署成功后，访问 http://localhost:4000 地址（默认端口），即可查看项目预览。
