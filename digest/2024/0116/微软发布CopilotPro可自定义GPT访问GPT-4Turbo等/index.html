微软发布CopilotPro：可自定义GPT、访问GPT-4Turbo等


  • By AiBard123
  • January 16, 2024 - 2 min read



作者： AIGC开放社区 来源： AIGC开放社区

专注AIGC领域的专业社区，关注微软&OpenAI、百度文心一言、讯飞星火等大语言模型（LLM）的发展和应用落地，聚焦LLM的市场研究和AIGC开发者生态，欢迎关注！

1月16日，微软在官网发布了——Copilot Pro。

Copilot Pro主要面向个人用户 ，提供更强的生成式AI功能，例如，创建自定义GPT，访问OpenAI最新模型 GPT-4 Turb，使用最新图像模型Dall-E 3创建各类精美图片，支持在安卓、iOS移动应用中使用等。

Copilot Pro可帮助个人用户在写作、营销、开发、生活、娱乐等各个领域，全面提升工作效率，每月订阅费用20美元。

Copilot Pro功能展示

Copilot Pro主要功能介绍

Copilot GPT： 这是Copilot Pro最重要的功能之一。目前，官方已经推出了健身、旅行、烹饪等特定领域的GPT助手。

很快，用户可借助Copilot GPT Builder，通过上传专有数据、指令优化等功能全程无需编写任何代码，就能创建独一无二的自定义GPT助手。

访问GPT-4 Turbo： 从今天开始，所有Copilot Pro用户都能访问OpenAI最新发布的模型GPT-4 Turb，即便是在高峰期也没问题。GPT-4 Turbo在创作、理解等方面更加优秀、可靠。

每天100次图像创作： Copilot Pro用户可以使用Designer 的 Image Creator（以前是Bing Image Creator）每天进行100次图像创作。

整体创建速度更快，同时图像质量获得了提升，也能创建横向图像。

Microsoft 365： Microsoft 365的个人和家庭订阅用户，可以在 PC、Mac 和 iPad 上访问 Word、PowerPoint、Outlook 和 OneNote 中的Copilot。

支持移动应用： 用户可以在安卓、iOS移动应用中使用Copilot Pro，功能与PC、网页版一样，同时数据互通可以查看其他端口上的使用情况。

Copilot for Microsoft 365更新

微软还对面向企业用户的Copilot for Microsoft 365进行了更新。

内测期间，40%的财富 100 强企业参与了微软的抢先体验计划，包括埃森哲、毕马威、普华永道、本田、Visa、BP等，帮助他们成功开启了“人机协作”的全新工作模式。

现在，微软不再限制Copilot for Microsoft 365的规模，任何大、中、小企业都能使用该产品，并且可以按需购买没有任何强制 。

1）适用于Microsoft 365的Copilot现在普遍适用于，拥有 Microsoft 365 商业高级版和商业标准版的小型企业。客户能以每人每月30美元的价格，购买1—299个席位。

2）微软取消了商业计划的300个席位的最低购买量，并使Copilot可供Office 365 E3 和 E5客户使用（之前需要 Microsoft 365 许可证）。

3）商业客户现在可以通过微软强大的Microsoft云解决方案提供商合作伙伴网络，购买 Copilot for Microsoft 365 。

4）所有符合教育教职员工以及18 岁以上的高等教育学生，都有使用 Microsoft 365 的资格。
